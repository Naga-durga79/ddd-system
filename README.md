📌 Problem Statement

Traditional learning platforms often suffer from low student engagement and motivation.
Students tend to lose interest due to lack of:

   Immediate feedback

   Progress visibility

   Reward mechanisms


This results in reduced learning effectiveness and poor performance tracking.


---

💡 Solution

The Dopamine Driven Dashboard (DDD) is a full-stack web application that applies gamification principles to enhance learning engagement.

It introduces:

🎯 Goal-oriented dashboards

🏅 Reward systems (badges)

📊 Real-time performance analytics

🔄 Instant feedback loops


👉 The system is designed to trigger motivation (dopamine effect) and encourage consistent learning behavior.

⚙️ Installation & Setup

1️⃣ Clone the Repository

git clone https://github.com/Naga-durga79/ddd-system.git
cd ddd-system


---

2️⃣ Backend Setup

cd backend
npm install
npm start


---

3️⃣ Frontend Setup

cd frontend
npm install
npm run dev


---

🔐 Environment Variables

Create a .env file inside backend:

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key


---

▶️ Running the Application

Backend → http://localhost:5000

Frontend → http://localhost:5173
